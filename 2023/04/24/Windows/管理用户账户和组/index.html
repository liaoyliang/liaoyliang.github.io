管理用户账户和组

liaoyl 发布于 2023-04-24

5.3.3 任务1 创建本地用户账户

本任务在win2008独立服务器上实验(先不加入域)，以administrator身份登录计算机

1.创建本地用户账户

(1)执行”开始”—“管理工具”—“计算机管理”操作，打开”计算机管理”框

(2)在上图窗口中，展开”本地用户和组”，在”用户”目录上右键单击，选择”新用户”选项，如图所示

(3)设置用户名、密码等信息，包括’账户已禁用’选项等，创建完单击”关闭，返回”计算机管理框”

5.3.2 任务2 设置本地用户账户的属性

用户账户不能只设置用户名和密码，还要设置用户隶属的用户组、用户配置文件、用户的拔入权限、终端用户设置等会，在”本地用户和组”，双击刚刚创建的student1用户，打开如图所示的对话框

1.”常规”选项卡

上图就是”常规”选项卡，可以设置用户有关的一些描述信息，包括全名、描述、账户选项等，管理员可以设置密码或禁用账户，也可以解除绑定

2.”隶属于”选项卡

可以设置将该账户加入到其他本地组中，新增的用户账户默认加入user组，user组用户一般不具备一些特殊权限，如安装应用程序、修改系统设置等，将”student1”添加到管理员用户组步骤如下

单击下图”添加”

在最底下对话框输入Power users,单击”检查名称”按钮，名称会变为WIN\Power Users(用户名\所属组)

如果不希望手动输入名称，也可单击”高级”按钮，再点击”立即查找”

，从列表中选择一个或多个组，如下图所示

3.”配置文件”选项卡

在”配置文件”可以设置用户账户的配置文件路径、登录脚本和主文件夹路径，默认用户配置文件位于”C:\user\default”下，该文件夹是隐藏文件夹，用户student1的配置文件在”C:\user\student1”下

4.登录脚本

登录脚本是用户登录计算机时自动运行的脚本文件，脚本文件的拓展名可以是VBS、BAT或CMD

其他选项卡(如拔入、远程控制选项卡)，请参考Windows server 2008的帮助文件

5.3.3 任务3 删除本地用户账户

在”计算机管理”控制台中，右键要删除的用户账户，可以执行删除功能，但是系统内置账户如Administrator、Guest等无法删除

5.3.4 任务4 使用命令行创建用户

创建新用户格式如下：

net user username password /add

例如要建立一个名为mike、密码为123_ABC的用户，可以使用命令：

net user mike 123_ABC /add

要修改旧账户的密码，可以按照如下步骤操作

1.打开”计算机管理”对话框

2.单击”本地用户和组”

3.右键要重设密码的账号，点击设置密码

或者使用命令行方式

net user username password

例如要将用户mike的密码设置为456_ABC,可以运行命令：

net user mike 456_ABC

5.3.5 任务5 管理本地组

1.要新建一个组，需先打开”计算机管理”对话框，右键单击”组”文件夹，选择”新建组”，在”新建组”对话框输入组名和描述。然后单击”添加”向组中添加成员

或者使用命令行方式，格式为：

net localgroup groupname /add

例如要新建一个名为sales的组，可以输入命令：

net localgroup sales /add

2.为本地组添加成员

可以将对象添加到任何组，在域中，这些对象可以是本地用户、域用户，甚至是其他本地组或组域，但是在工作组环境中，本地组的成员只能是用户账户

为了将成员mike添加到本地组common，可以执行以下操作

(1)打开”开始”—“管理工具”—“计算机管理”对话框

(2)展开本地用户和组，选择组，然后双击所要选择的组common

(3)单击添加，选择用户即可

或者使用命令行，格式如下：

net localgroup groupname username /add

例如要将用户mike加入Administrators组中，可以使用命令：

net localgroup Administrators mike /add

5.3.6 任务6 管理域用户

1.域用户账户

2.创建域用户账户

(1)以域管理员身份登录win2008-1，打开”开始”—“管理工具”—“Active Directory用户和计算机”，展开域，首先创建用户和密码

(2)右键Users容器，新建用户，如图所示，在其中输入姓、名，系统可以自动填充完整姓名

(3)选择想要实现的密码选项，单击下一步，单击完成，左键窗格中的Users容器，显示用户列表，双击想要配置的用户，可以进行诸多配置

(4)当添加多个用户账号时，可以以一个设置好的用户账号作为模板

，右键要复制的模板账户，选择复制，即可复制账号的所有属性，提高效率

3.验证域账户用户

验证yangyun域用户是否能在win2008-2计算机(已加入域xwx40.com)上登录

注销用户，选择”切换用户”—“其他用户”，用户名填yangyun，密码：输入建立该域用户的密码

5.3.7 任务7 管理域中的组账户

1.使用win2008-1打开”Active Directory用户和计算机”，展开域，右键Users，新建组

同理，创建common组

2.认识常用的内置组

3.为组sales指定成员

(1)以管理员身份登录win2008-1，打开”Active Directory用户和计算机”对话框，选择Users选项，右键要添组成员的组”sales”，选择”成员”

(2)单击”添加”，打开如图对话框

(3)单击”对象类型”，打开”对象类型”，如图所示，选择”计算机”和”用户”复选框，单击确定返回

(4)单击”位置”，打开’位置’对话框，选择在xwx40.com域中查找，如图所示

(5)单击”高级”，打开如图对话框，单击”立即查找”，列出所有用户和计算机账户，选择yangyun

(6)单击确定，所选计算机或用户将被添加到组

(7)验证结果

4.将用户添加至组

(1)以管理员身份登录win2008-1，打开”Active Directory用户和计算机”对话框，选择Users选项，右键要添加到组的用户yangyun

(2)输入所要加入的组名称，中间用;隔开，也可点击”高级“进入高级选项，点击”立即查找”，列出所有用户组

5.查看用户组sales属性

（1）打开”Active Directory用户和计算机”对话框，选择Users，右键所要选择的用户组sales，打开sales用户组的属性对话框，选择”成员”选项，显示用户组sales所拥有的所有计算机和用户账户

(2)打开”Active Directory用户和计算机”对话框,选择Users，右键所要选择的用户yangyun，打开yangyun用户的属性对话框，选择”隶属于”选项，显示该用户属于的所有用户组
