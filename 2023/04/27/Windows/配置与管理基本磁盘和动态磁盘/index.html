配置与管理基本磁盘和动态磁盘

liaoyl 发布于 2023-04-27

项目7 配置与管理基本磁盘和动态磁盘

7.3.1 任务1 管理基本磁盘

1.新建基本卷

(1)打开”计算机管理”—“磁盘管理”，右键未分配的卷，新建简单卷

(2)根据引导，设置卷的大小为500MB

(3)单击下一步，分配驱动器号

(4)单击下一步，选择格式文化的文件系统，选择NTFS

(5)下一步，完成新建简单卷，建立完成主分区

2.创建拓展分区

windows server 2008磁盘管理不能直接创建拓展分区，必须先创建完三个主分区才可以创建拓展磁盘分区，window三个主分区以后才是拓展分区

(1)本例已有三个主分区，无需再创建，若没有三个主分区，自行创建即可

(2)完成三个主分区创建后，在该磁盘未分区空间右键”新建简单卷”

(3)后面过程与创建主分区相似，不同的是当创建完成，显示”状态良好”的分区信息后，系统自动将刚才这个分区设置为拓展分区的一个逻辑驱动器

3.指定活动的磁盘分区

计算机启动时会启动被设为”活动”的磁盘分区

假设第一个磁盘分区装的是windows server，第二个磁盘分区装的是Linux，第一个磁盘分区被设置为”活动”，则会启动windows server

设置方法：右键磁盘1主分区C，选择”将分区标为活动分区”选项即可指定活动磁盘分区

4.更改驱动器号和路径

(1)在”磁盘管理”中，右键单击目标驱动器E，在弹出的快捷菜单中选择”更改驱动器号和路径”，如图所示

(2)单击”更改”，可以更改驱动器号；单击”添加”，打开”添加驱动器号或路径”对话框

(3)输入完成后，单击确定即可

7.3.3 任务3 建立动态磁盘卷

1.以管理员身份登录，将磁盘转换为动态磁盘(若磁盘处于脱机状态，右键该磁盘联机即可)

2.在未分配空间上右键选择”新建RAID-5卷”选项，打开”新建卷向导”

3.单击下一步，打开”选择磁盘”对话框，如图所示，选择要创建的RAID-5卷需要使用的磁盘，选择空间容量为1021MB，对于RAID-5卷来说，至少要选择3个以上动态磁盘（大小也要一样）

4.为RAID-5指定驱动器号和文件系统类型，完成向导设置

5.建立完成的RAID-5卷如图所示

建立其他类型动态 卷方法与此类似，右键磁盘未分配空间，出现选项选择即可

7.3.4 任务4 维护动态卷

1.维护镜像卷

(1)在显示未”丢失”或”脱机”的磁盘右键单击删除镜像，如果出现故障的镜像卷成员位于要故障的控制器上，则在有故障的控制器上安装新的磁盘不能解决问题

(2)使用新磁盘替换损坏磁盘

(3)右键要重新镜像的卷(不是已删除的卷)，然后在弹出的快捷菜单中选择”添加镜像”，打开如图所示的”添加镜像”对话框，选择合适的磁盘，单击”添加镜像”，系统会用新的磁盘重建镜像

2.维护RAID-5

(1)在”磁盘管理”右键单击想要修复RAID-5卷，选择”重新激活卷”选项

(2)由于卷成员磁盘失效，所以会弹出”缺少成员”信息，单击确定

(3)再次右键单击要修复的RAID-5卷，选择修复卷

(4)在”RAID-5修复卷”对话框中，选择新添加的动态磁盘，然后单击确定

(5)在磁盘管理器中，可以看到RAID-5在新磁盘上重新建立，并进行数据同步，同步完成后，RAID-5卷的故障被修复成功

7.3.5 任务5 管理磁盘配额

1.设置磁盘配额

(1)在”磁盘管理”对话框中，右键要启用磁盘配额的磁盘卷，然后点击属性

(2)选择”配额”选项卡，如图所示

(3)选择”启用配额管理”复选框，然后为新用户设置磁盘空间限制数值

(4)若要对原有的用户设置配额，单击”配额项”，打开如图所示的窗口

(5)选择”配额”—“新建配额项”，或单击工具栏上的”新建配额项”

打开”选择用户”对话框

单击”高级”按钮，再单击”立即查找”

即可在”搜索结果”列表中选择当前计算机用户，并设置磁盘配额，关闭配额项窗口

(6)回到”配额”选项卡，如果需要限制受配额影响的用户使用超过配额的空间，则选择”拒绝将磁盘空间给超过配额限制的用户”，单击确定即可

7.3.6 任务6 整理磁盘碎片

删除和添加文件会在磁盘产生”磁盘碎片”，影响访问磁盘时间，降低计算机性能

解决方法：以此单击 “开始”—“附件”—“系统工具”—“磁盘碎片整理程序”

打开”磁盘碎片整理程序”窗口

一般情况下，选择要进行磁盘碎片整理的磁盘后，要分析磁盘分区状态，单击”分析磁盘”，可以对所选磁盘分区进行分析

分析完毕后会打开对话框询问是否对磁盘进行碎片整理，若要整理，单击”磁盘碎片整理”即可
